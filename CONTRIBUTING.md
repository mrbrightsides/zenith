## üõ†Ô∏è Development Workflow

1.  **Fork the Project:** Create your own fork of the [mrbrightsides/zenith](https://github.com/mrbrightsides/zenith) repository.
2.  **Create a Feature Branch:** `git checkout -b feature/amazing-modality`
3.  **Code & Style:** 
    *   We use **React 19** and **Tailwind CSS**.
    *   Maintain the "Glassmorphism" and "Futuristic" design language.
    *   Ensure all new Gemini API calls are wrapped in appropriate error handling (see `GeminiService.ts`).
4.  **Commit Changes:** `git commit -m 'Add some amazing modality'`
5.  **Push to Branch:** `git push origin feature/amazing-modality`
6.  **Open a Pull Request:** Describe your changes in detail and link to any relevant issues.
